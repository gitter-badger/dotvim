Installation

RO:
git clone git://github.com/saamalik/dotvim.git ~/.vim

RW:
git clone git@github.com:saamalik/dotvim.git ~/.vim

Create symlinks:

ln -s ~/.vim/vimrc ~/.vimrc

All Steps:

git clone git://github.com/saamalik/dotvim.git ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
cd ~/.vim
git submodule init
git submodule update


MinTTY looks prettier with the CONSOLA font, in the fonts directory. Install it and restart mintty.
