## Installation

*RO*:  
git clone git://github.com/saamalik/dotvim.git ~/.vim

*RW*:  
git clone git@github.com:saamalik/dotvim.git ~/.vim

Install:
```
$ ./sync-bundles.sh
```


MinTTY looks prettier with the CONSOLA font, in the windows fonts directory. Install it and restart mintty.
